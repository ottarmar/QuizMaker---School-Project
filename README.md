# QuizMaker---School-Project
Project from school, where we worked as a team to make a quiz maker. 
There you can make a quiz and send it to students to take. 
The application is in split into two, 
the first half is for the admin/teacher to make a quiz and see statistics of the quizzes already taken. 
The other half is for the students to see quizzes they can take and get results from finished quiz.
